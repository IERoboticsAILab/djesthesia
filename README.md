# Welcome to DJESTHESIA
Welcome to the DJESTHESIA! DJESTHESIA is a inovative interface for DJs that explores the integration of Tangible User Interfaces (TUIs). We aim to transform the DJ into both a performer and a performance with DJESTHESIA. 

## Four Interaction modes
1. **Knob changes the music** <br> DJ changes the music by physically turning knobs.
2. **Music changes visuals**<br> the visual projected onto the DJ working space will change depending on the low-latency audio.
3. **Gesture changes visuals**<br> Motion Capture(MoCap) system track gecture of hands with MoCap marker glove that interacts with the generated visuals.
4. **Gesture changes music**<br> With two-handed festures, a DJ can draw EQ curves that will be applied to the music. <br>
